## 💡 CHI TIẾT KỸ THUẬT

### Date Input Component
```javascript
// Khởi tạo tự động
initDateInputs(); // Tìm tất cả .date-input-vn

// Format khi nhập
formatDateInput(input); // Tự động thêm /

// Validate khi blur
validateAndConvertDate(input); // Kiểm tra ngày hợp lệ

// Lấy giá trị ISO
const isoDate = getISODate(input); // yyyy-mm-dd

// Set giá trị
setDateValue(input, '2025-02-10'); // Hiển thị 10/02/2025
```

### Booking Form Modes
```javascript
// URL Patterns
booking-form-v3.html              → CREATE mode
booking-form-v3.html?id=1&mode=view → VIEW mode (read-only)
booking-form-v3.html?id=1         → EDIT mode

// Detect mode
checkMode() {
  const id = urlParams.get('id');
  const mode = urlParams.get('mode');
  
  if (id && mode === 'view') setViewMode();
  else if (id) setEditMode();
  else setCreateMode();
}
```

## 🎨 LUỒNG HOẠT ĐỘNG

### Create Flow (Tạo mới)
```
1. Click "Tạo phiếu thuê phòng"
   ↓
2. booking-form-v3.html (no params)
   ↓
3. checkMode() → CREATE mode
   ↓
4. Nhập thông tin (date auto-format)
   ↓
5. Click "Lưu phiếu"
   ↓
6. createBooking() → Save to DB
   ↓
7. Redirect → booking-list-v3.html
```

### View Flow (Xem)
```
1. Click icon 👁️
   ↓
2. booking-form-v3.html?id=1&mode=view
   ↓
3. checkMode() → VIEW mode
   ↓
4. loadBookingData(1)
   ↓
5. setViewMode() → Disable all inputs
   ↓
6. Hiển thị read-only
```

### Edit Flow (Sửa)
```
1. Click icon ✏️
   ↓
2. booking-form-v3.html?id=1
   ↓
3. checkMode() → EDIT mode
   ↓
4. loadBookingData(1)
   ↓
5. setEditMode() → Title + button text
   ↓
6. Sửa thông tin
   ↓
7. Click "Cập nhật phiếu"
   ↓
8. updateBooking() → Update DB
   ↓
9. Redirect → booking-list-v3.html
```
